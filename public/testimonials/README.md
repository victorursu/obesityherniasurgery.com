# Testimonials Images

This folder should contain the before and after photos for testimonials.

## Image Naming Convention

- `before-1.jpg` through `before-17.jpg` - Before photos
- `after-1.jpg` through `after-17.jpg` - After photos
- `placeholder.jpg` - Placeholder image (optional, for missing images)

## Adding Images

1. Download the images from the original site
2. Name them according to the convention above
3. Place them in this directory

The testimonials data is stored in `data.json` and will automatically load the images when they are added to this folder.
